# 英语第一季排产表

## 目标

第一季的目标不是立刻变现，而是尽快验证三件事：

1. 什么标题会有人点
2. 什么内容会有人听下去
3. 什么表达会有人留言反馈

因此第一季的标准不是“做得最全”，而是“尽快上线，尽快拿反馈，尽快学到东西”。

## 第一季定位

- 语言：英语
- 集数：8 集
- 形式：长音频为主，短视频持续引流，长视频只做高潜力集
- 调性：现代成长型内容，不做学术讲解，不做空泛灵性
- 核心卖点：古老智慧解释现代焦虑、关系、节奏、决策

## 发布策略

- 每周至少 1 集长音频
- 每周 3-5 条短视频
- 第一季只选 2-3 集做完整长视频
- 第一季结束后，再决定是否同步做西语

## 8 集选题

### EP01

- Title: `The Source of Everything`
- Core Promise: 为什么你越想控制人生，越容易失衡
- Core Hook: `What if the reason you feel stuck is not that life is against you, but that you're pushing at the wrong moment?`
- Priority: High
- Long video: Yes

### EP02

- Title: `Balance Is Not 50/50`
- Core Promise: 平衡不是平均分配，而是识别当下需要什么
- Core Hook: `Most people think balance means doing everything evenly. It doesn't.`
- Priority: High
- Long video: Yes

### EP03

- Title: `Why Anxiety Feels Like Lost Rhythm`
- Core Promise: 焦虑不只是压力，也可能是节奏错位
- Core Hook: `Anxiety is not always fear. Sometimes it's misalignment.`
- Priority: High
- Long video: Maybe

### EP04

- Title: `When to Push, When to Wait`
- Core Promise: 决策问题很多不是能力问题，而是时机判断问题
- Core Hook: `The hardest decisions are often not about what to do, but when to do it.`
- Priority: High
- Long video: Maybe

### EP05

- Title: `Why Forcing Love Breaks Connection`
- Core Promise: 关系里过度用力，往往会破坏连接
- Core Hook: `Too much force breaks connection. Too much retreat creates distance.`
- Priority: High
- Long video: Yes

### EP06

- Title: `Burnout Is an Argument With Rhythm`
- Core Promise: 倦怠很多时候不是你不行，而是你违背了节奏
- Core Hook: `Burnout is what happens when ambition stops listening to rhythm.`
- Priority: Medium
- Long video: No

### EP07

- Title: `Clarity Is Not Control`
- Core Promise: 真正的清晰，不来自更用力，而来自更对齐
- Core Hook: `Control feels powerful, but clarity is quieter.`
- Priority: Medium
- Long video: No

### EP08

- Title: `How to Live Through Change Without Panic`
- Core Promise: 学会在变化中保持稳定，而不是在变化中失控
- Core Hook: `Change is not the problem. Panic is.`
- Priority: Medium
- Long video: Maybe

## 推荐生产顺序

第一批先做：

1. EP01
2. EP03
3. EP05

原因：

- 这 3 集最容易和现代痛点直接连接
- 最容易切短视频
- 最容易获得评论和反馈

第二批再做：

1. EP02
2. EP04
3. EP06

第三批收尾：

1. EP07
2. EP08

## 每集最低交付标准

每集不是都要满配。

最低交付：

- 1 份英文定稿
- 1 条完整长音频
- 3 条短视频

高配交付：

- 1 份长视频视觉脚本
- 1 条完整长视频

## 第一季成功标准

达到以下任一项，就算方向跑通：

- 有自然评论出现
- 有人听完后留言说“这说的是我”
- 某条短视频明显拉来新点击
- 某一集的完播率明显高于其他集
- 有人主动问下一集什么时候出

## 第一季结束后的决策点

第一季结束后只回答 3 个问题：

1. 用户最吃哪种问题表达？
2. 用户更喜欢关系、焦虑、节奏，还是决策？
3. 哪几集值得做西语同步？

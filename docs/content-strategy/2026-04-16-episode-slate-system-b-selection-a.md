# The Outcome Code 前 8 集主线

## 使用原则

这 8 集是当前默认的第一季前半主线。

它们的任务不是“讲完整个《易经》”，而是：

- 先打进英语平台
- 先建立清晰的问题入口
- 先验证用户最吃哪一类判断

## EP01

### Title

`Why Smart People Keep Making Bad Timing Decisions`

### Core Role

旗舰开场集。

### Primary Source Assets

- `01_阴阳之道_0722a4ea`
- `易经的智慧15待机之道_c60fea13`

### Supporting Source Assets

- `易经的智慧14需要等待_360be6a2`

### Real Problem

很多聪明人不是不努力，而是在错误的节奏上不断加力。

### Pattern Judgment

真正让人失手的，往往不是能力不够，而是误把用力当成了判断。

### Result Promise

让用户学会区分：

- 什么时候该推
- 什么时候该停
- 什么时候你自己正在制造问题

### Length Target

`10-12 分钟`

### Best Public Use

- 长视频/音频双发
- 3 条强 Shorts
- 后续所有内容的入口母题

### Pairable Pattern Case

- `Trump, Force, and the Collapse of Balance`

### Pairable Radar Trigger

- 公众人物的过度扩张
- 组织内部因节奏误判而翻车

## EP02

### Title

`When More Force Makes You Lose`

### Core Role

第一批里最强的冲突型标题。

### Primary Source Assets

- `易经的智慧16争强好胜_8f930193`

### Supporting Source Assets

- `95决其当决_5756c833`

### Real Problem

很多人遇到阻力的本能反应就是更用力，但结果常常更差。

### Pattern Judgment

力量失去调节之后，会从优势变成负债。

### Result Promise

帮助用户看懂：

- 什么时候硬推有用
- 什么时候硬推会毁掉局面

### Length Target

`10-12 分钟`

### Best Public Use

- 标题强
- 评论潜力高
- 很适合切成争议型 Shorts

### Pairable Pattern Case

- 高冲突公众人物
- 商业谈判或公关硬碰硬案例

### Pairable Radar Trigger

- 任何“升级冲突”的热搜事件

## EP03

### Title

`What Anxiety Is Really Telling You`

### Core Role

把“时机/节奏/错位”翻译成最容易共鸣的现代情绪语言。

### Primary Source Assets

- `113_52艮卦上_大忧大定_av31579235_p45_45c132b9`

### Supporting Source Assets

- `114_52艮卦下_止其所止_av31579235_p46_1bb3133b`

### Real Problem

很多人以为焦虑只是情绪问题，其实很多焦虑来自节奏错位和长期失衡。

### Pattern Judgment

焦虑常常不是危险本身，而是你跟当下节奏失去同步后的身体信号。

### Result Promise

让用户用更实用的框架读懂：

- 焦虑
- 赶路感
- 长期过度紧张

### Length Target

`10-14 分钟`

### Best Public Use

- 女性受众和年轻专业人群更容易接入
- 适合音频
- 也适合做视觉化长视频

### Pairable Pattern Case

- 高压职业人群
- 创业者或名人的疲劳型失衡

### Pairable Radar Trigger

- 公众人物情绪崩溃
- 高压行业集体焦虑话题

## EP04

### Title

`Knowing When to Stop Is a Superpower`

### Core Role

把“止”的智慧转成非常现代的边界、自控与止损主题。

### Primary Source Assets

- `114_52艮卦下_止其所止_av31579235_p46_1bb3133b`

### Supporting Source Assets

- `135_60节卦上_调节得宜_av31579235_p67_94e0e8d7`

### Real Problem

现代人普遍更会开始，不会停止；更会推进，不会收手。

### Pattern Judgment

真正的强，不是永远往前，而是知道什么时候停下，避免自己越界。

### Result Promise

帮助用户识别：

- 什么时候继续努力
- 什么时候其实是在过度消耗
- 什么时候必须止损

### Length Target

`8-11 分钟`

### Best Public Use

- 非常适合做 Shorts
- 适合引向后续 premium 内容

### Pairable Pattern Case

- 名人因不收手而翻车
- 企业过度扩张案例

### Pairable Radar Trigger

- “越做越错”的舆论或商业事件

## EP05

### Title

`Why Slow Progress Beats Fast Collapse`

### Core Role

把“循序渐进”翻译成英语世界很容易接受的成长框架。

### Primary Source Assets

- `115_53渐卦上_循序渐进_av31579235_p47_274a501d`

### Supporting Source Assets

- `116_53渐卦下_悠久稳健_av31579235_p48_68f62ad0`

### Real Problem

很多人急于求成，反而让本来能稳住的事情变得更脆弱。

### Pattern Judgment

长期有效的上升，通常都比看起来慢，但也更不容易崩。

### Result Promise

让用户学会看清：

- 稳定增长和虚假提速的区别
- 为什么很多“快”其实是在埋雷

### Length Target

`9-12 分钟`

### Best Public Use

- 职业成长
- 关系推进
- 创作者成长

### Pairable Pattern Case

- 爆红后迅速沉没的创作者/品牌

### Pairable Radar Trigger

- 某个快速起势后失速的组织或人物

## EP06

### Title

`Why Success Gets Harder After You Win`

### Core Role

这是“后成功时代”的第一主题。

### Primary Source Assets

- `141_63既济卦上_守成艰难_av31579235_p73_cd4f7b5e`

### Supporting Source Assets

- `142_63既济卦下_福过灾生_av31579235_p74_161737c5`

### Real Problem

很多人会拼命争取成功，却不会经营成功之后的局面。

### Pattern Judgment

成功不是终点，而是新的风险入口。

### Result Promise

让用户理解：

- 为什么赢了以后更容易犯错
- 为什么高点最需要 restraint

### Length Target

`10-13 分钟`

### Best Public Use

- 创业者
- 管理者
- 有明显事业焦虑的人群

### Pairable Pattern Case

- 上位后失控的公司/名人/领导者

### Pairable Radar Trigger

- “刚赢就开始输”的商业或公众事件

## EP07

### Title

`How to See a Breakdown Before It Happens`

### Core Role

把“见微知著”做成非常有传播力的预判主题。

### Primary Source Assets

- `43见微知著_c5985307`

### Supporting Source Assets

- `40临事而惧_d51f6ec8`

### Real Problem

很多局面不是突然坏掉的，而是大家太晚才看见信号。

### Pattern Judgment

结果往往在小信号、行为漂移、节奏变化里已经开始显形。

### Result Promise

让用户学会在事情还小的时候就看懂：

- 哪些迹象不能忽视
- 哪些地方已经在变坏

### Length Target

`10-12 分钟`

### Best Public Use

- 标题吸引力强
- 很适合做长视频
- 很适合做评论区互动

### Pairable Pattern Case

- 组织衰退
- 关系破裂
- CEO 或名人公信力下滑

### Pairable Radar Trigger

- 任何“其实早有迹象”的热点

## EP08

### Title

`Why Wanting Something Right Now Can Ruin the Outcome`

### Core Role

这一集负责把“需要、欲望、耐心、时机”做成更贴近现代生活的语言。

### Primary Source Assets

- `易经的智慧14需要等待_360be6a2`

### Supporting Source Assets

- `易经的智慧15待机之道_c60fea13`

### Real Problem

很多问题不是因为我们想要得太多，而是想要得太快。

### Pattern Judgment

真正毁掉结果的，常常不是目标，而是对目标失去节奏感。

### Result Promise

帮助用户分辨：

- 合理的需要
- 过早的索取
- 过度的急切

### Length Target

`8-11 分钟`

### Best Public Use

- 关系
- 事业
- 金钱
- 欲望管理

### Pairable Pattern Case

- 关系推进过快
- 创业扩张过快
- 个人野心失速

### Pairable Radar Trigger

- 任何“太想要所以做坏了”的实时事件

## 当前最先上线的 3 集

### 第 1 批上线优先级

1. `EP01`
2. `EP02`
3. `EP03`

原因：

- 三集共同形成一条清晰入口线：
  - 时机
  - 力量失衡
  - 焦虑与错位
- 标题都很像英语平台已经被证明有效的问题句
- 很容易拆 Shorts
- 也很容易带出 radar / case 内容

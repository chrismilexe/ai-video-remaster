# 第一批选题方案 A

## 方案名

`Broad Entry Batch`

## 选择逻辑

这版的目标不是最“像《易经》”，而是最容易先做出英语平台能吃的入口。

优先标准：

1. 标题一眼就能翻成现代英语问题
2. 不需要用户先懂中国传统概念
3. 能自然落到 `timing / pressure / burnout / relationships / outcomes`
4. 最适合做 evergreen 主集和 Shorts

## 现在不要优先的资产

以下内容先降权：

- 起源神话和体系说明：`121_龙马负图`、`122_神龟载书`
- 过重的方法论内部说明：`133_卦中有卦`、`134_自占自解`、`45卦序之道`、`67_易经体例`
- 过于“先解释传统文本”的素材

原因：
这些内容可以以后做，但不适合当第一批英文入口。

# 香港公司入驻方案详解

**适用对象**: 香港公司 + 中国国籍董事/股东  
**目标平台**: ACX/Audible, Spotify, Apple Books, YouTube  
**最后更新**: 2026-03-27

---

## ✅ 可行性结论

**完全可行**。香港公司可直接入驻所有主流平台，中国国籍不构成障碍。

### 核心依据
- 平台审核**公司注册地**和**税务身份**，不审核董事国籍
- 香港非美国税收协定国，但作为独立税务管辖区被认可
- 中美税收协定（版权收入）适用10%预扣税率

---

## 🏢 公司架构方案

### 当前架构（推荐）
```
香港公司
├── 董事: 中国护照持有者（你）
├── 股东: 你 或 + 合伙人
├── 注册地址: 香港
├── 银行账户: 香港本地银行
└── 税务身份: 香港税务居民
```

### 合伙人角色（可选增强）
| 合伙人身份 | 可选用途 | 必需性 |
|-----------|---------|--------|
| 香港身份 | 本地联系人、银行开户辅助 | 可选 |
| 美国身份 | 美国银行账户、 faster USD receipt | 可选 |

**结论**: 两种身份均为"锦上添花"，非必需。

## 📋 平台入驻详情

### 1. ACX / Audible ✅ 直接入驻

**税务设置**:
1. 访问 https://tax.audible.com
2. 创建账户（使用香港公司邮箱）
3. 选择 "Foreign Entity"
4. 填写 W-8BEN-E（公司版W-8BEN）

**W-8BEN-E 关键字段**:
| 字段 | 填写内容 |
|------|---------|
| Line 1 | 香港公司全称 |
| Line 2 | 公司注册国家: Hong Kong |
| Line 3 | Entity type: Corporation |
| Line 6a (FTIN) | 香港商业登记证号 (BRN) 或 "not legally required" |
| Line 8 | 香港地址 |
| Line 9 | 声明香港税务居民身份 |
| Treaty Claim | 中美税收协定 Article 12 (版权) |

**预扣税率**:
- 默认: 30%
- 填写W-8BEN-E后: **10%**（版权收入）

**收款设置**:
- 支持香港银行账户（汇丰/中银/万里汇）
- 最低提现: $10
- 付款周期: 月度，延迟约30天

- 列支敦士登、卢森堡、荷兰、新西兰、瑞士

**香港❌不在列**

**解决方案**:

#### 方案A: 通过聚合商（推荐）
**INaudio** (原Findaway Voices)
- 支持香港公司
- 费用: $25/年 或 15%分成
- 自动分发到 Spotify + Apple + 其他平台
- 聚合商地址在美国，绕过地理限制

**Author's Republic**
- 支持香港公司
- 费用: 20%分成（无年费）
- 适合测试阶段

#### 方案B: Distribution API（2026新功能）
- 可在Acast/Libsyn托管
- 同时接入Spotify变现
- 无需更换托管平台

**收款币种**: USD / HKD
**最低提现**: $100

## 💰 税务优化策略

### 预扣税对比

| 收入类型 | 无协定 | 中美协定 | 香港公司适用 |
|---------|--------|---------|-------------|
| 版权收入（有声书） | 30% | **10%** | ✅ 适用 |
| 服务收入 | 30% | 无 | 30% |

**关键点**: 必须在Tax Interview中声明税务协定优惠。

### 香港本地税务

**利得税（Profits Tax）**:
- 税率: 16.5%
- **离岸收入免税**: 非香港来源的收入可申请豁免
- 有声书收入通常被视为离岸（平台在境外，听众在全球）

**申请离岸豁免**:
- 首年审计时声明
- 保留业务性质证明（合同、平台协议等）
- 每3年复审一次

**建议**: 咨询香港注册会计师，确保合规。

## 🏦 银行账户方案

### 方案1: 香港本地银行（推荐）

**汇丰 One (HSBC One)**
- 优点: 国际品牌，接受度高
- 开户: 线上申请，1-2周
- 费用: 无月费，低汇率差价
- 多币种: 支持USD/HKD/EUR/GBP

**中银香港**
- 优点: 中资背景，人民币便利
- 开户: 需预约，支持内地身份证
- 费用: 较低

**所需文件**:
- 公司注册证书 (CI)
- 商业登记证 (BR)
- 公司章程 (AA)
- 董事身份证/护照
- 地址证明

### 方案2: 虚拟银行（快速）

**Wise Business**
- 优点: 开户快（1-3天），多币种账户
- 费用: 0.3-0.5%汇率差价
- 缺点: 部分平台可能不接受

**Airwallex / 万里汇**
- 优点: 专为跨境电商设计
- 费用: 较低
- 支持: 大部分平台接受

### 方案3: 美国账户（可选增强）

如需美国账户:
1. 合伙人美国身份开设 **Mercury** 或 **Wise US**
2. 香港公司作为股东/受益人
3. 注意: 美国账户需申报FBAR（如> $10,000）

**推荐优先级**: 香港汇丰 One > 万里汇 > 美国账户

## 📝 分步入驻指南

### Phase 1: 准备（1-2周）

1. **确认公司状态**
   - [ ] 年审有效（查NAR1）
   - [ ] 商业登记证未过期
   - [ ] 有最新周年申报表

2. **开设银行账户**
   - [ ] 申请汇丰One（线上）
   - [ ] 或申请万里汇（更快）

3. **准备税务文件**
   - [ ] 香港商业登记证（BR）扫描件
   - [ ] 公司注册证书（CI）
   - [ ] 董事护照/身份证

### Phase 2: ACX入驻（2-3周）

1. **注册账户**
   - 访问 acx.com
   - 选择 "Rights Holder"
   - 使用香港公司邮箱注册

2. **税务面试**（关键步骤）
   - 访问 tax.audible.com
   - 创建独立账户（不要用Amazon登录）
   - 完成W-8BEN-E
   - 确认预扣税显示为10%

3. **绑定银行**
   - 添加香港账户信息
   - 等待验证（可能需银行对账单）

4. **上传内容**
   - 上传音频文件
   - 提交Retail Sample
   - 选择Non-exclusive
   - 等待审核（10-14工作日）

### Phase 3: 扩展分发（1-2周）

1. **注册聚合商**
   - 推荐: Author's Republic（无upfront cost）
   - 或使用INaudio（$25/年）

2. **上传分发**
   - 使用同一套音频文件
   - 选择目标平台: Spotify, Apple, Google Play等
   - 等待审核（各平台3-10工作日）

### Phase 4: YouTube（并行）

1. **创建频道**
   - 使用Google账号（建议新建，与AdSense一致）
   - 设计频道Banner/Logo

2. **申请AdSense**
   - 访问 google.com/adsense
   - 选择Business Account
   - 填写香港公司信息
   - 等待PIN验证

3. **满足YPP门槛**
   - 1,000订阅
   - 4,000小时观看 或 10M Shorts views

## ⚠️ 风险控制

### 常见拒绝原因

1. **税务面试未完成**
   - 症状: 无法设置银行信息
   - 解决: 重新访问tax.audible.com完成

2. **音频质量不达标**
   - 症状: ACX审核拒绝
   - 解决: 检查RMS、Peak、Room Tone

3. **银行验证失败**
   - 症状: 提现失败
   - 解决: 确认账户名与注册名完全一致

### 合规检查清单

- [ ] 内容标注"AI-generated"（如使用AI语音）
- [ ] 避免医疗/治疗声明
- [ ] 保留所有平台协议副本
- [ ] 记录收入来源和日期
- [ ] 香港公司按时年审

- 平台预扣10%
